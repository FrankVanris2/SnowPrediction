# Installation

Install Flask:
```
pip install Flask
```

If not have install npm by installing node.js:
```
https://nodejs.org/en/download
```

Install packages:
```
npm i
```



Build the client:
```
npm run build
```

# Run server
```
python -m flask --app server run
```

# Client development

Run the watch command:
```
npm run watch
```

# Clean up
```
npm run clean
```
